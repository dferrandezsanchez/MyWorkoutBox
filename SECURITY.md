# Security Policy

## Supported Versions

MyWorkoutBox is currently in alpha. Security fixes are applied to the latest version on `main` and the latest published release only.

## Reporting A Vulnerability

Report vulnerabilities through [GitHub private vulnerability reporting](https://github.com/dferrandezsanchez/MyWorkoutBox/security/advisories/new).

Do not open a public issue and do not include real credentials, personal data, or production data in a report.

Include:

- The affected endpoint, component, or version.
- Reproduction steps or a minimal proof of concept.
- Expected impact and required preconditions.
- Any suggested mitigation, when available.

Reports will be reviewed before details are made public. No guaranteed response or remediation timeline is offered while the project remains independently maintained.
